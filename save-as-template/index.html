Problem Statement
Users refine proposals on real jobs — adjusting line items, restructuring sections, tweaking terms — but have no way to carry those improvements back into their templates. The only workarounds are manually rebuilding the template or starting from scratch. Good work on proposals is a dead end. Save as Template adds a new action to the proposal editor's quote-details page, letting users overwrite the source template or create a new one — with permission-aware branching for owners vs. standard users, and optional nudges when sending or leaving a scratch proposal.

Resolved decisions
Permission denied — "Save as Template" hidden entirely for users without PROPOSALS_TEMPLATES_CREATE. They only see the linked template in the sidebar.
Duplicate names — Allowed. No validation needed.
Long template name — System truncates, full name on tooltip hover.
Pre-send scope — Only shows for proposals created without a template. Once per proposal.
